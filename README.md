# MindWell — Mental Health & Wellness Tracker

**Job 43** | **Package:** com.jnetai.mindwell | **Repo:** https://github.com/jnetai-clawbot/MindWell

## What It Does
A mental health and wellness tracking app. Log daily moods, journal entries, and self-care activities. Visualizes mood trends over time and provides wellness reminders.

## Features
- Daily mood logging with emoji scale
- Journal/diary entries with timestamped notes
- Self-care activity tracker (exercise, meditation, sleep, social)
- Mood trend visualization
- Wellness reminders and prompts
- Data export as JSON

## Build
- Android API 26+ (Android 8.0)
- Kotlin, Room database, ViewBinding + XML
- Material3 Dark theme
- GitHub Actions CI with release APK
